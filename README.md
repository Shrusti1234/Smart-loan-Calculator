# Smart-loan-Calculator
Smart-loan-Calculator-description

Smart Loan Calculator is a simple and responsive web application built using HTML, CSS, and JavaScript.  
It helps users calculate their **monthly loan payment**, **total payment**, and **total interest** based on the loan amount, interest rate, and loan duration.

## Features
- Calculates monthly EMI
- Calculates total payment and total interest
- Handles zero-interest loans
- Input validation for correct values
- Clean and responsive UI
- Instant calculation on button click

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
smart-loan-calculator/
├── index.html
├── style.css
└── script.js

## How to Run
1. Download or clone the project
2. Open `index.html` in a browser
3. Enter loan amount, interest rate, and years
4. Click **Calculate Loan**
5. View the loan summary instantly

- JavaScript event handling
- DOM manipulation
